hexo next Mist主题页脚居中

发表于 2024-04-01 分类于 hexo

前言

搭好hexo博客后，感觉Muse主题不太好看，换了Mist主题。但是这个主题的页脚不居中，看着很别扭。所以想设置页脚居中。网上查了一下，查到的方法没有作用。最后折腾了一会，终于搞定了。

解决方法

打开/themes/next/source/css/_common/outline/footer目录下的index.styl文件。找到.footer_inner{...}，把text-align: ...改成text-align: center。最后执行命令：

hexo clean
hexo g
hexo d
